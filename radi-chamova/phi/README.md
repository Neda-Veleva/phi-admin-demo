# PHI Admin

Нова администрация за `phi.bg`, изградена като самостоятелно Vite + React + TypeScript приложение.

## Какво има вътре

- Секция `Обучения`
- Под-секция `Списък с обучения`
- Под-секция `Заинтересовани хора`
- Бележки към всеки човек
- Seed съдържание, вдъхновено от `phi.bg`
- Локално съхранение чрез `localStorage`, така че приложението работи веднага
- `supabase/schema.sql` за следваща стъпка към реална база

## Стартиране

```bash
npm install
npm run dev
```

## Данни

Демо данните се пазят в `localStorage` под ключ `phi-admin-store-v1`.
Ако искаш чист старт, изтрий този ключ от браузъра.

## Следваща стъпка

След импортиране на [`supabase/schema.sql`](./supabase/schema.sql) може да се подмени `src/lib/persistence.ts` с Supabase CRUD слой по модела от предишните CMS проекти.
